A high-performance, visually stunning personal portfolio designed with a focus on premium aesthetics and smooth user experiences. Built entirely with **Vanilla JavaScript, HTML5, and CSS3**, this project showcases advanced front-end techniques including custom shaders, interactive canvases, and complex animations.

---

## 💎 Features & Highlights

### 🎨 Visual Excellence
- **Cosmic Cursor Canvas**: A custom-built, physics-based cursor with comet trails and lock-on reticles.
- **Legendary Background**: High-performance particle systems and gradient morphing for a "legendary" UI feel.
- **Glassmorphism UI**: Modern, sleek cards with dynamic tilt effects and holographic overlays.
- **Ambient Glow Systems**: Section-specific ambient lighting that adapts as you scroll.

### 🧠 Advanced Interaction
- **3D Perspective Cards**: Interactive project and skill cards that respond to mouse movement with smooth 3D tilt.
- **Smart Navbar**: Intelligent section tracking and a smooth mobile-responsive menu.
- **Dynamic Content**: Data-driven architecture for projects, internships, and achievements for easy updates.

### ⚡ Performance & Design
- **Zero Frameworks**: Lightweight and fast, achieving high performance by using direct DOM manipulation and native APIs.
- **Responsive Layout**: Pixel-perfect design across all devices — from desktop monitors to mobile phones.
- **Accessibility**: Includes smooth scroll progress indicators and respects user preferences for reduced motion.

## 📁 Project Architecture

```
portfolio/
├── index.html              # Core HTML structure
├── assets/
│   ├── images/             # Profile, project thumbnails & textures
│   ├── audio/              # Ambient background music
│   └── fonts/              # Custom typography (Inter & Outfit)
├── css/
│   ├── variables.css       # Design tokens & color system
│   ├── base.css            # Premium foundation & global resets
│   ├── enhancements.css    # Advanced visual effects (Glow, Particles)
│   ├── animations.css      # Keyframes & reveal utilities
│   └── responsive.css      # Fluid media queries
├── js/
│   ├── main.js             # Core application logic & initialisation
│   ├── legendary-bg.js     # Cosmic particle canvas 
│   ├── animations.js       # Magnetic interactions & custom effects
│   └── stats-counter.js    # Adaptive number counting engine
└── data/                   # JSON-like data stores for easy management
```

---

## 🛠️ Tech Stack

- **Core**: HTML5, CSS3, JavaScript (ES6+)
- **Graphics**: HTML5 Canvas API, SVG
- **Animations**: CSS Transitions, Keyframes, RequestAnimationFrame
- **Typography**: Google Fonts (Inter, Outfit)
- **Deployment**: Local dev server (npm/python)

## 🚀 Getting Started

1. **Clone the repository**:
   ```bash
   git clone https://github.com/Naveenkumar-s-24/My-Portfolio.git
   ```
2. **Launch the site**:
   Simply open `index.html` in your browser. For the best experience (including local asset loading), use a local server:
   ```bash
   npm install && npm start
   # OR
   npx serve .
   ```
